NULL Pointer Bugs

A NULL character pointer is NOT the same as a pointer to a NULL string. In Solaris, the following program will lead to a "Segmentation Fault".

  1. #include <stdio.h>
  2. #include <string.h>
  4. int main()
  5. {
  6.     char *string=NULL;
  8.     printf("/n String length=%d",strlen(string));
  10.     return 0;
  11. }
